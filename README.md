# ERA5 Weather Visualizer

Interactive climate data visualization for the Indian Subcontinent using ERA5 reanalysis data.

## Tech Stack

**Frontend:** React, Vite, Tailwind CSS, Leaflet, Chart.js

**Backend:** Flask, xarray, SQLAlchemy

## Run Locally

### Backend

```bash
cd backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python app.py
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

Open http://localhost:5173
